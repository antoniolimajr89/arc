Arc [![Build Status][travis-image]][travis-url]
===========

Parser de Markdown para AFC escrito em Node.js

### Instalação
  
  1. Substitua os valores em *setup-ws.sh* pelos valores reais
  2. *source setup-ws.sh*
  3. *make build*
